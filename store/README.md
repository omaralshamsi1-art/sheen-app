# SHEEN — App Store Listing Assets

Everything needed to submit SHEEN to the App Store and Google Play (text + graphics).

| File | What it is |
|---|---|
| `app-store-listing.md` | Apple: name, subtitle, description, keywords, review notes |
| `google-play-listing.md` | Play: title, short/full description, store checklist |
| `data-safety-and-privacy.md` | Apple App Privacy + Google Data Safety answers |
| `screenshots.md` | Screenshot capture plan + required sizes |
| `feature-graphic.png` | Play feature graphic, 1024×500 (ready to upload) |

cover camera, notifications, biometrics, and Apple/Google Pay).
